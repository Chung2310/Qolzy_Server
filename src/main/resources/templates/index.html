Giới thiệu

Xin chào! Mình là Chung Đỗ, hiện đang là sinh viên ngành Kỹ thuật Máy tính tại Trường Kỹ thuật Công Nghiệp - Đại học Thái Nguyên. Mình đam mê lập trình và phát triển phần mềm, đặc biệt là các ứng dụng di động và hệ thống Back-end.

Trong quá trình học tập, mình đã tích lũy kinh nghiệm phát triển ứng dụng trên Android Studio, từ việc thiết kế giao diện đến triển khai logic nghiệp vụ. Đồng thời, mình cũng xây dựng các API Back-end sử dụng Java Spring Boot để phục vụ các ứng dụng và dịch vụ web.

Mình luôn hứng thú với việc học hỏi công nghệ mới, tìm hiểu các framework hiện đại, và tham gia vào các dự án thực tế để nâng cao kỹ năng lập trình. Mục tiêu của mình là trở thành một Full-stack Developer có khả năng phát triển ứng dụng từ front-end đến back-end, đồng thời áp dụng tốt các chuẩn công nghiệp và best practices trong lập trình.

Công nghệ và kỹ năng đã sử dụng:

  • Android Studio (Mobile App Development)
  • Java Spring Boot (Back-end API)
  • Git / GitHub (Version Control)
  • SQL / MySQL (Database Management)
